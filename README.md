### 这是什么
> 这是一个简单的在CocosCreater 3.8.x中使用Rapier2d物理引擎的样例

### 兼容版本
` 3.6.x以上上都支持 ,因为是用的wasm库` `"@dimforge/rapier2d-compat": "^0.15.0"`

### 使用注意
1. 拉取仓库
2. 进入根目录后 `npm install`
